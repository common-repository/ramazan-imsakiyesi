== Description ==

Bu eklenti, Ramazan ayı boyunca sitenizde, il il İmsak (Sahur), Güneş, Öğle, İkindi, Akşam (İftar), Yatsı saatlerini gösterir.
Eklenti hazırlanırken, Türkiye Cumhuriyeti Diyanet İşleri Başkanlığı resmi internet sitesinden alınan veriler kullanılmıştır.

== Installation ==

1. ramazan-imsakiyesi klasörünü wp-content/plugins klasörüne yükleyiniz.
2. Eklentiler sayfasından eklentiyi etkinleştiriniz.
3. Görünüm/Bileşenler sayfasından Ramazan İmsakiyesi bileşenini istediğiniz bir alana sürükleyip bırakın.

== Screenshots ==

1. Bileşen, Görünüm/Bileşenler sayfasına yerleşiyor.
2. Bileşenin sitede görünümü.
